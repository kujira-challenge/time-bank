1. Time Bankとは

Time Bankは、チームメンバーの時間貢献を記録・可視化するための総合的な管理システムです。 個人やチームの作業時間を「時間銀行」として記録し、分析することで、プロジェクトの進捗管理や メンバーの貢献度を明確にすることができます。

🎯 Time Bankの目的

  • 透明性の確保: チーム全員の時間貢献を可視化
  • 公平な評価: 客観的なデータに基づく貢献度の把握
  • プロジェクト管理: タグやタスクによる作業分類と分析
  • チーム協働: Wiki的な相互編集による柔軟な記録管理

主な特徴

特徴 説明
シンプルな時間記録 直感的なUIで簡単に作業時間を記録
リアルタイム集計 ダッシュボードで即座に時間データを分析
タグによる分類 プロジェクトや作業内容でエントリを分類
代理入力機能 他のメンバーの代わりにエントリを作成可能
相互編集 全員がすべてのエントリを編集・修正可能（Wiki方式）
CSVエクスポート データをExcelなどで詳細分析

2. はじめに - ログインと初期設定

1アカウントについて

💡 招待制システム

Time Bankは招待制のため、管理者があらかじめアカウントを作成する必要があります。 アカウントが必要な場合は、管理者にご連絡ください。

2初回ログイン

  1. 管理者から提供されたURLにアクセス
  2. 「ログイン」ボタンをクリック
  3. メールアドレスを入力
  4. 初回の場合は「パスワードを忘れた場合」からパスワードリセットを実行
  5. メールで届いたリンクから新しいパスワードを設定
  6. 設定したパスワードでログイン

⚠️ パスワードの管理

パスワードは安全な場所に保管してください。忘れた場合は「パスワードを忘れた場合」からリセットできます。

3ログイン後の画面

ログインに成功すると、ホーム画面が表示されます。ホーム画面には以下のメニューが表示されます：

🏠 ホーム画面のメニュー

  • エントリ一覧: 記録した時間を確認・管理
  • エントリ作成: 新しい時間を記録
  • ダッシュボード: 時間の集計と分析
  • タスク依頼: タスクを依頼・管理
  • ユーザー管理: 貢献者を追加・管理
  • ヘルプ: 使い方ガイド
  • マイページ: 自分のプロフィール

3. 主要機能

3.1 エントリ作成・管理

Time Bankの中核機能です。作業時間をエントリとして記録し、チーム全体で共有できます。

エントリ作成の手順

  1. ホーム画面またはナビゲーションから「エントリ作成」をクリック
  2. 以下の情報を入力：
    • 貢献者: エントリを記録する対象者を選択。自分または他のメンバーを選択可能（代理入力機能）
    • 活動日 必須: 作業を行った具体的な日付を選択
    • 時間数 必須: 作業した時間を入力（0.5時間単位、0〜100時間）
    • タグ: 作業内容を分類するタグを追加（複数選択可、最大10個）
      よく使うタグがボタンで表示され、クリックで簡単に追加できます：
      • 業務効率化プロジェクト
      • 時間銀行プロジェクト
      • 自分発見塾
    • メモ: 作業内容の詳細や特記事項（最大1000文字）
  3. 「作成」ボタンをクリック

✨ タグ入力の便利機能

  • デフォルトタグ: よく使うタグがボタンで表示され、クリックするだけで追加可能
  • オートコンプリート: 過去に使用したタグが自動的に候補として表示
  • カスタムタグ: 新しいタグを自由に入力して追加可能

エントリ一覧の使い方

「エントリ一覧」画面では、記録されたすべてのエントリを確認・管理できます。

📋 フィルタリング機能

  • 週開始日でフィルタ: 特定の週のエントリのみ表示
  • タグでフィルタ: 特定のタグが付いたエントリのみ表示
  • 貢献者でフィルタ: 特定のメンバーのエントリのみ表示
  • 自分のエントリのみ表示: 自分が作成したエントリに絞り込み

エントリの編集と削除

🤝 相互編集機能

全員がすべてのエントリを編集・削除できます（Wiki方式）。 これにより、チームメンバー同士で協力して記録の精度を高めることができます。

  • 各エントリの「編集」ボタンから内容を修正可能
  • 「削除」ボタンで不要なエントリを削除可能
  • 信頼できるチーム環境を前提とした設計

代理入力機能の活用

👥 代理入力とは

代理入力機能により、他のメンバーの代わりにエントリを作成できます。 例えば、ミーティングや会議の記録係が、参加者全員分の時間を一括で記録する際に便利です。

使用例：

  • 会議の参加者全員の時間を幹事が一括記録
  • 体調不良などで記録できないメンバーの代わりに入力
  • チームメンバーではないが時間記録が必要なユーザーの代理入力

💡 ログインしないユーザーについて

一部のユーザーは、チームメンバーではないものの、貢献時間の記録のために登録されている場合があります。 そのようなユーザーはログインできないため、他のメンバーが代理入力で記録を行います。

このようなユーザーの追加が必要な場合は、管理者にご連絡ください。

3.2 ダッシュボード

ダッシュボードでは、記録された時間データを集計・分析し、グラフや表で可視化します。

表示される情報

📊 サマリー統計

  • 総記録時間: チーム全体の累計作業時間
  • 平均評価: フィードバック評価の平均値
  • 評価件数: 受け取った評価の総数
  • 価値スコア: 時間と評価を組み合わせた総合指標
    計算式: 時間 × 1.0 + 評価 × 2.0

週次推移グラフ

過去12週間の時間推移を折れ線グラフで表示。チームの活動状況の変化を一目で確認できます。

タグ別時間分布

各タグ（プロジェクト）ごとの時間配分を円グラフまたは棒グラフで表示。 どのプロジェクトに多くの時間が使われているかを把握できます。

チームランキング

メンバー別の時間数と価値スコアをランキング形式で表示。 公平な評価とモチベーション向上に役立ちます。

CSVエクスポート機能

📥 当月データのダウンロード

ダッシュボードの「当月CSVエクスポート」ボタンをクリックすると、 当月のエントリデータをCSV形式でダウンロードできます。

活用方法：

  • Excelで開いて詳細な分析
  • 月次報告書の作成
  • 外部ツールとの連携
  • データのバックアップ

3.3 タスク管理

チームメンバー間でタスクを依頼・管理する機能です。

タスクの作成

  1. 「タスク依頼」ページから「タスク作成」をクリック
  2. タスクの詳細を入力：
    • タスク名
    • 説明
    • 担当者
    • 期限
  3. 「作成」ボタンをクリック

タスクの管理

タスク一覧では、以下の操作が可能です：

  • タスクの進捗状況を確認
  • タスクの詳細を表示
  • フィードバックを確認
  • タスクのステータスを更新

3.4 ユーザー管理

チームメンバーの情報を確認・管理する機能です。

ユーザー一覧

登録されているすべてのユーザーが一覧表示されます。各ユーザーの情報には以下が含まれます：

  • 表示名
  • メールアドレス
  • ロール（member / admin）
  • アクティブ状態

💡 新しいユーザーの追加

新しいメンバーやログインしないユーザーの追加は、管理者が行います。 ユーザーの追加が必要な場合は、管理者にご連絡ください。

3.5 マイページ

自分のプロフィール情報を確認・編集できます。

表示される情報

  • 表示名
  • メールアドレス
  • 登録日
  • 自分の登録履歴

4. 活用のヒント

⏰ 定期的に記録しましょう

週に一度、または作業が完了したタイミングで時間を記録すると、正確な時間管理ができます。 毎週金曜日や月曜日など、決まった曜日に記録する習慣をつけるのがおすすめです。

🏷️ タグを効果的に活用しましょう

チームで共通のタグを使うことで、プロジェクト別の時間配分が明確になります。

  • プロジェクト名: 業務効率化プロジェクト、時間銀行プロジェクトなど
  • 作業種別: 開発、デザイン、ミーティング、レビューなど
  • 優先度: 緊急、重要、通常など

📝 メモを詳しく残しましょう

具体的な作業内容をメモに残すことで、振り返りや報告に役立ちます。

記載例：

  • 「ユーザー認証機能のバグ修正（パスワードリセットフローの改善）」
  • 「月次報告書の作成とレビュー依頼」
  • 「チームミーティング：来週のタスク優先順位の決定」

📊 ダッシュボードを定期的に確認しましょう

定期的にダッシュボードを確認することで、以下のメリットがあります：

  • チーム全体の活動状況を把握
  • 自分の貢献度を客観的に確認
  • プロジェクトの進捗を可視化
  • 時間配分の偏りに気づける

🤝 相互編集機能を活用しましょう

チームメンバー同士で記録の精度を高めることができます：

  • 誤字脱字や計算ミスを見つけたら修正
  • 不明確な記録があれば本人に確認して補足
  • タグの統一や整理に協力

5. よくある質問

Q1: エントリを間違えて作成してしまいました

A: 「エントリ一覧」から該当のエントリを見つけて、「編集」ボタンで修正するか、 「削除」ボタンで削除できます。相互編集機能により、すべてのエントリが編集・削除可能です。

Q2: 活動日はどのように選べばいいですか？

A: 作業を実際に行った日付を選択してください。 具体的な日付を記録することで、より正確な時間管理が可能になります。

Q3: 価値スコアとは何ですか？

A: 価値スコアは、記録時間とフィードバック評価を組み合わせた総合指標です。

計算式: 時間 × 1.0 + 評価 × 2.0

例えば、10時間の作業で評価5.0を受けた場合：
10 × 1.0 + 5.0 × 2.0 = 20.0

Q4: CSVエクスポートは何に使えますか？

A: ダッシュボードの「当月CSVエクスポート」ボタンから、 月次データをCSV形式でダウンロードできます。

活用例：

  • Excelで開いて詳細な分析
  • 月次報告書の作成
  • 外部ツールとの連携
  • データのバックアップ

Q5: 他のメンバーのエントリを編集してもいいですか？

A: 相互編集機能により、技術的にはすべてのエントリを編集できますが、 以下のガイドラインを守ってください：

  • 誤字脱字や明らかなミスの修正は自由に行ってOK
  • 内容に関わる大きな変更は、該当メンバーに確認を取ってから
  • 削除する場合は特に慎重に判断
  • チームの信頼関係を大切にする

Q6: ログインしないユーザーのエントリはどうやって作成しますか？

A: エントリ作成画面で「貢献者」のドロップダウンから該当するユーザーを選択し、 通常通りエントリを作成してください。ログインしないユーザーは、時間記録のためだけに登録されています。

新たにそのようなユーザーの追加が必要な場合は、管理者にご連絡ください。

Q7: タグは何個まで設定できますか？

A: 1つのエントリに最大10個のタグを設定できます。 デフォルトタグと過去に使用したタグがオートコンプリートで表示されるので、 積極的に活用してください。

Q8: パスワードを忘れてしまいました

A: ログイン画面の「パスワードを忘れた場合」リンクをクリックし、 登録したメールアドレスを入力してください。パスワードリセットのリンクが送信されます。

6. トラブルシューティング

症状 原因と対処方法
ログインできない
  • メールアドレスとパスワードが正しいか確認
  • 初回の場合は「パスワードを忘れた場合」からパスワード設定
  • アカウントが作成されているか管理者に確認
ログイン後に画面が空白になる
  • ブラウザのキャッシュをクリア
  • 別のブラウザで試す
  • 問題が続く場合は管理者に連絡
データが表示されない
  • ページを再読み込み（F5キー）
  • ログアウト後、再度ログイン
  • 問題が続く場合は管理者に連絡
エントリが作成できない
  • 必須項目（活動日、時間数）が入力されているか確認
  • 時間数が0〜100の範囲内か確認
  • ネットワーク接続を確認
  • エラーメッセージが表示されている場合は管理者に報告
CSVエクスポートができない
  • ブラウザのポップアップブロックを無効化
  • ダウンロードフォルダの容量を確認
  • 別のブラウザで試す

⚠️ 問題が解決しない場合

上記の方法で解決しない場合は、以下の情報を添えて管理者に連絡してください：

  • 発生している症状の詳細
  • エラーメッセージ（表示されている場合）
  • 使用しているブラウザとバージョン
  • 操作の手順（再現方法）
